I build backend systems that are clean, fast, and built to last. Not just code that works — code that's maintainable, optimized, and ready to scale when the load doubles.

I care about the things that don't show up in demos: query performance, proper architecture, clean abstractions, and systems that don't fall apart under real traffic. I've worked on large-scale products with real data, real users, and real consequences — that shapes how I think about every line I write.

My environment is Arch Linux, my weapons of choice are Laravel and Spring Boot, and I don't commit until it's clean.
